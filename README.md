
# SymfonyUXDemo



## Getting started

 - Go to google api console interface to enable a Youtube API and get a Key
 - Get the project
 - Change the key in the .env file
 - Run 
 

>   make build
>   make up
>   make composer
>   make asset-map

 - Open your browser on https://localhost
 - Bypass security message
 - Enjoy
